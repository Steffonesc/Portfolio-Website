Fitted App

This app was designed to help people who didn't have the time to go to the gym and had a busy schedule So this app will help people get a full workout from the comfort of their own home.

  • Role: UI Designer
  • Course: CareerFoundry UX Design Program
  • Tools: Axure, Balsamiq, Figma, and Invision
  • Duration: 2 Months

Objective

Motivate people to workout anywhere, anytime, and also any skill level

The 5 W's

Who: People who are new or returning to fitness and want to find workouts they can do anywhere and anytime.

What: A responsive web/mobile app that users can pick, search, and also favorite workouts. An app where users can keep their progress and set daily goals for themselves.

When: The app can be used at anytime. Users will use the app when they are searching, choosing, and following a plan.

Where: Users can use the app anywhere they please. As long as they have an open space and they are logged onto the app.

Why: To live a healthy lifestyle. So the user can feel better physically and emotionally.

User Personas

The user persona on who this app was designed for and make sure her needs are met.

User Flows

With the user flows provided in the brief, the user flows were created to focus on the persona and make sure every task is met.

User Stories

  • As a new user, I want to learn about different exercises, so that I can figure out what is best for me.
  • As a new user, I want to be shown how the exercises are done, so that I know I’m doing them correctly.
  • As a frequent user, I want to be able to schedule exercises for working out, so that I build positive habits.
  • As a frequent user, I want to be able to earn achievements or rewards, so that I can stay motivated.
  • As a frequent user, I want to complete daily challenges, so that I can have an additional way to stay motivated.
  • As a frequent user, I want to track progression and record what I’ve done, so that I can see my progress over time.
  • As a frequent user, I want to be able to share routines with my friends who may also be interested, so that I can encourage them to become healthier.
